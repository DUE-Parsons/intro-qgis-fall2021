Using Open Data

  1. Open opendata.cityofnewyork.us.
  2. Search for 311 and select the top link, 311 Service Requests from 2010 to Present.
  3. Click View Data to preview the data. Don't download the entire file right now since it is very large.
  4. Click the Filter tab.
  5. Change the condition to Created Date, is after and use the date picker to select 10/1/2021.
  6. Add a new filter condition, set it to Complaint Type, contains. For the value I used Noise but feel free to explore other complaint type values like Rodent.
  7. Go to the Export tab and download a CSV (the top link is the one you want).
  8. Don't worry about using it yet, but hold on to it for the next exercise.
